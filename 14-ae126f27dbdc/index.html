博客备份

Author: 夜七笔记 Date: 四月 1, 2025  21:57:52 Category: 个人随笔

最近感觉我这个电脑可能要出问题了，现在已经出现了《win11版本已终止服务》这个提示了，附图：

现在是怎么更新也不行了，就算是使用官方的方法也是不行，提示我电脑配件太老了，已经不支持win11的运行了。

所以想着万一哪天电脑彻底开不了机的时候，我这个小笔记站点就彻底不能更新了，那就悲剧了。

趁着现在还能开机，就尝试着把hexo的主体文件备份一下。

我的备份方法是：把hexo的主体文件备份到GitHub仓库里。就算是以后我这台电脑彻底不行的时候，也能在GitHub里拉取hexo的主体文件，不至于让这个小博客彻底的灰飞烟灭。

记录一下备份hexo到GitHub的命令符

echo “# nilincc” >> README.md

  • git init
  • git add README.md
  • git commit -m “first commit”
  • git branch -M main
  • git remote add origin git@github.com:名称/仓库名.git
  • git push -u origin main

后期在写完博文之后，在敲几个命令更新一下hexo的主体文件

  • git status # 查看文件状态
  • git add –all # 添加所有更改后的文件
  • git commit -m “你要commit的内容” #对这些文件进行 commit
  • git push # 同步到 Github 云端仓库

以后一定要养成备份的习惯，因为未来谁也说不准会出现什么不可预测的事情。
